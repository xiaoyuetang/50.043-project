# Developer Guide
##### Setup Steps
1. cd keelia-search
2. export FLASK_APP=app.py
3. export FLASK_DEBUG=1
4. flask run

##### Search Steps
1. Type in search bar (I have changed it to a big one)
2. The input will be passed to search-result page
3. Look into the app.py to edit to connect to database
